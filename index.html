Javascript (JS)

an object-oriented computer programming language commonly used to create interactive effects within web browsers

Operators

a symbol that is used to perform an operation (assignment, comparison, arithmetic, etc.)

Variables

can be thought of as named containers; data can be placed into these containers and then refer to the data simply by naming the container; must be declared before using

Variable Naming Conventions

camelCase; all names start with a letter

Functions

a procedure: a set of statements that performs a task or calculates a value; must be defined somewhere in the scope from which you wish to call it

Methods

actions that can be performed on objects; functions stored as object properties

Arguments

a value that is passed between programs or functions; independent items, or variable, that contain data or codes

Parameters

a special kind of variable, used in a subroutine to refer to one of the pieces of data provided as input to the subroutine

Return

where a variable or other information goes back to the subroutine

Chaining Methods

invoking multiple method calls in object-oriented programming languages; each method returns an object, allowing calls to be chained together in a single statement without requiring variables to store the intermediate results

Strings

a finite sequence of characters

Booleans

a data type having two values: true or false

Undefined

a variable has been declared but has not yet been assigned a value

NaN

not-a-number is a weirdo global obect returned when some mathematical operation failed

Escape

this funciton computes a new string in which certain characters have been replaced by a hexadecimal escape sequence

Alert

a pop up in the browser

Comments

important information written in the code to help others understand what is going on

jQuery

a write less, do more Javascript library

Attributes

provide additional information about an element; always specified in the start tag
